Clinical Definition¶

The purpose of this notebook is to create a multiple logistic regression model to predict MDRO status using composite features based on antibiotic resistance. The features used to create the model are shown below.

Dummy variables were created by grouping antibiotics into the following categories:

  • Resistant to Meropenem OR Imipenem
  • Resistant to Ertapenem
  • Resistant to Ceftazidime
  • Resistant to Gentamicin OR Tobramycin
  • Resistant to Cefepime
  • Resistant to Pip-Tazo
  • Resistant to Ceftriaxone

Compute Antibiotic Weights¶

Antibiotic resitance score is computed as $$R = \sum_{i=1}^{n}x_i w_i$$ where $x_i$ is 0 if organism is sensitive to antibiotic $i$ and 1 if organism is resistant to antibiotic $i$. $n$ is the total number of antibiotics text (18 in this case) and $w_i$ is the weight for antibiotic $i$ and is calculated such that stronger antibiotics recieve larger weights and weaker antibiotics recieve smaller weights. Antibiotic strength is calculated as the fraction of susceptible organism as follows $$w_i = 1 - Q_i$$ where $Q_i$ represents the fraction of organisms resistant to antibiotic $i$.

Create Composite Features¶

Composite features are based on antibiotic resistance. For example if an organism is resistant to meropenem OR imipenem then the feature will be TRUE. Further more the dependent variable is created by discretizing the organisms resistant scores using a threshold value of 5. All organisms with score greater than 5 are considered MDRO. The dependent variable is called mdro. Here we use an antibiotic resistance score of 5 or greater to designate an MDRO...this classification becomes the dependent variable on which we train our MLR model.

Using $\beta$ estimates from MLR to create clinical definition¶

I use the $\beta$ estimates from the model created after dropping ceftriaxone. A point system was created created by taking the aboslute value of the $\beta$ estiamtes and rounding to the nearest whole number. We can now calcualte a clinical score for each organism by summing the total number of points depending on whether the organism is resistant or susceptible for each antibotic or group of antibiotics. For example, suppose an organism is resistant to cefepime and gentamycin, we would calculate the clinical score as follows $$score = 4 + 3 = 7$$. This score exceeds the threshold of 6 and therefore we would classify it as an MDRO. The treshold we use here (6) is loosly derived from the intercept of the MLR model, however due to rounding of the $\beta$ coefficients it is not exactly the same as the intercept and was selected based on the ROC analysis below. Basically, the threshold of 6 was selected as it provides a good balance between sensitivity and specificity. Later, I test the performance of our model using various possible threshold in the ROC analysis.

Applying the clinical definition¶

In this section I create a function that direcly implements the clinical definition (i.e. uses the point system above) to calculate clinical scores based on organisms resistance pattern. This function allows you to set the weights (we use the ones derived/specified in the previous table as defaults) and a threshold (we use 6 as the default). We can then calculate a clinical score by passing an example resistance pattern to the function

In [11]:
# Function used to create predictions based on clinical definition
clinical_prediction <- function(data, weights = c(1, 3, 3, 4, 5, 4), thresh=6){
    score = as.matrix(subset(data, select = c(piperacillin_tazobactam,
                                      meropenem_imipenem,
                                      cefepime,
                                      ertapenem,
                                      ceftazidime,
                                      gentamicin_tobramycin))) %*% weights
    res = data.frame(score = score, class = (score > thresh))
    res$class = factor(res$class, levels = c('TRUE', 'FALSE'))
    res
}
