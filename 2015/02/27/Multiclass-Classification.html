Machine Learning: Multi-class classification with logistic regression

Machine Learning: Multi-class classification with logistic regression

2015-02-27 22:51:00 +0100

This exercise implements a one-vs-all linear regression classifier to recognize handwritten digits using gradient descent. Regularized gradient descent was introduced in the post Regularized Logistic Regression. Once trained, the model was able to recognize handwritten digits with an accuracy of 83.8%. It is anticipated that the accuracy could be improved by allowing more complete convergence of the cost function.
